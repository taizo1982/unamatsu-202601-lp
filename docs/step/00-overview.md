# LP制作ワークフロー概要

このプロジェクトはHTML&CSSからSSG形式のLPを制作するためのテンプレートです。

## ステップ一覧

| ステップ | ファイル | 説明 |
|---------|---------|------|
| 1 | [01-setup-design.md](01-setup-design.md) | デザインファイルの配置 |
| 2 | [02-setup-env.md](02-setup-env.md) | 環境変数の設定 |
| 3 | [03-optimize-images.md](03-optimize-images.md) | 画像の最適化 |
| 4 | [04-build-production.md](04-build-production.md) | 本番ビルド |
| 5 | [05-pagespeed-check.md](05-pagespeed-check.md) | PageSpeed対応チェック |

## 使い方

各ステップのプロンプトをClaude Codeにコピー&ペーストして実行してください。

```
例: 「01-setup-design.mdの内容を実行して」
```
